# Wey Backend API 文档

## 账户相关接口 (Account)

### 邮箱激活
- **URL**: `/account/activateemail/`
- **Method**: GET
- **参数**:
  - email: 用户邮箱
  - id: 用户ID
- **描述**: 激活用户邮箱
- **返回**: 激活状态信息
